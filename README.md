# Inception

## Description

This project is a hands-on, complete project designed to learn Docker and Docker compose, as well as, web technology architecure and the basics of:
* nginx with php-fpm
* wordpress
* mariadb.

### Goal

* Create a complete server capable of hosting wordpress with a db efficiently and securely

### Design choices

* In my project, a Dockerfile sets-up the container with the basic modules and configuration and the entrypoint script handles more detailed set up

### Virtual Machine vs Docker

| | Virtual machine | Docker |
| -- | -- | -- |
| OS | Complete OS | Shares OS kernel with host |
| Memory usage | High | Low |
| Hardware | Virtualized | Shared |
| Software Isolation | Can't access host OS | Blocks access to internet - Host files |

## Instructions

* Install docker - docker compose - make if not installed yet
* Check docker status (```sudo systemctl status```) and start it if not active with ```sudo systemctl start docker```
* Run the command ```make``` at the project root
* Visit ```localhost``` at a web browser

* To stop the containers, run ```make down``` at the project root
* To delete all data generated by the project run ```make clean``` at project root
